## Install

```sh
git clone https://github.com/peguimasid/42-Printf ft_printf
```

## Usage

1. Create a `main.c` file inside project `ft_printf` folder

```c
#include "ft_printf.h"
#include <stdio.h>

int	main(void)
{
	ft_printf("Hello %d | %s", 42, "Rio");
}
```

## Run

```sh
 make -s && cc main.c ./libftprintf.a && ./a.out && rm ./a.out && make -s fclean
```
